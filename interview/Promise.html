    # Promise 中 then 的第二个参数和 catch 有什么区别

    • reject 是用来抛出异常的，catch 是用来处理异常的
    • reject 是 Promise 的方法(Promise.reject())，而 then 和 catch 是 Promise 实例的方法(Promise.prototype.catch())
    • 如果 then 的第一个函数抛出异常，后面的 catch 可以捕获到，而 then 的第二个函数捕获不到
